# 2026-04-22 - ClawPass Desktop Session

## Summary
Great productive session - fixed multiple issues in the ClawPass Tauri desktop app.

## Issues Fixed

### 1. Save Entry button broken
- **Root cause**: Frontend was sending pre-encrypted passwords, but backend expects plaintext and encrypts internally
- **Fix**: Updated `addEntry` in `vault.ts` to send plaintext password/notes; backend handles encryption

### 2. Categories not showing
- **Root cause**: Frontend never called `loadCategories()`, categories sidebar not implemented
- **Fix**: Added `loadCategories()` on mount, category list loop in VaultView, category dropdown in AddEntryModal

### 3. Update Entry broken
- **Root cause**: Sending encrypted data directly instead of decrypting first
- **Fix**: `updateEntry` now decrypts stored password/notes before sending to backend

### 4. "All Items" and "Favorites" showing as user categories
- **Fix**: Hardcoded these as special system categories in sidebar; filter them out from user-managed list

### 5. Favorites category not filtering
- **Fix**: Special handling for `selectedCategory === '2'` to filter `is_favorite === true`

## Icons Changed
